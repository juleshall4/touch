# touch

Touch is a fun project designed for tracking hand placement when practicing touch typing.

## Tech stack

- React 19 and TypeScript
- TanStack Start, Router, and AI
- Tailwind CSS 4
- Vite and Vitest
- Biome and Ultracite
- Python, OpenCV, and YOLO vision tooling

## Commands

- `bun run dev` — run the app
- `bun run build` — build the app
- `bun run test` — run tests
- `bun run check` — check formatting and code quality
- `bun run fix` — fix formatting and code quality
